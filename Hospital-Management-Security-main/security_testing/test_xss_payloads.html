üîß Quick Testing Steps

Step 1: Test search.php

  2. In the search field, enter: <script>alert('XSS')</script>
  3. Submit the form
  4. Before fix: Alert box will appear
  5. After fix: Text will be displayed as-is

Step 2: Test contact.php

  1. Navigate to contact.html
  2. In the Name field, enter: <img src=x onerror=alert('XSS')>
  3. Fill other fields normally
  4. Submit the form
  5. Check where contact data is displayed (admin panel)
  6. Before fix: Script executes when viewing
  7. After fix: HTML is escaped and displayed as text

Step 3: Test URL parameters

  1. Craft URL: prescribe.php?fname=<script>alert('XSS')</script>&lname=Test&pid=1&ID=1&appdate=2025-01-01&apptime=10:00:00
  2. Navigate to this URL
  3. Before fix: Script in URL executes
  4. After fix: Parameter is sanitized/escaped
